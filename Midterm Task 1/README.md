# Midterm Lab Task 1 - Data Cleaning and Preparation using Excel

- Fit Column and row width and height
- TRIM extra spaces
- Remove NULL values
- Remove Duplicates

- Perform Table split using 2NF
- Perform 3 NF
- Mapped the Normalized tables as a Physical Data Model
